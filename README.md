# 🎯 관심사 뉴스 크롤러

12개 카테고리별로 오늘 날짜의 기사와 블로그를 크롤링하여 웹으로 보여주는 애플리케이션입니다.

## 📂 지원 카테고리

1. **건강·운동** - 다이어트, 자세교정, 통증관리, 루틴
2. **맛집·레시피** - 혼밥, 간단요리, 다이어트식, 지역맛집
3. **재테크·돈관리** - 절약, 가계부, 투자 입문, 소비습관
4. **여행·주말나들이** - 국내 당일치기, 코스 추천, 숨은 명소
5. **연애·관계·심리** - 대화법, 갈등 해결, MBTI/성향
6. **자기계발·공부법** - 시간관리, 집중, 독서, 자격증
7. **IT·앱·AI 트렌드** - 유용한 앱, 생산성 툴, AI 활용법
8. **뷰티·패션·그루밍** - 스킨케어, 스타일링, 남녀 공통 관리
9. **집·인테리어·살림** - 정리정돈, 청소템, 공간 꾸미기
10. **취미·문화생활** - 영화/드라마, 전시, 음악, 취미추천
11. **육아·가족·반려동물** - 육아팁, 가족관계, 반려동물 케어
12. **스포츠** - 최신 스포츠 뉴스

## ✨ 기능

- 📂 **카테고리 선택** - 원하는 관심사 선택
- 🔍 **실시간 크롤링** - 네이버 블로그, 다음 블로그, 티스토리에서 수집
- 📝 **기사 요약** - 자동 요약 생성
- 🖼️ **이미지 표시** - 각 기사의 이미지와 타이틀, 요약 표시
- 🎨 **현대적인 웹 UI** - 반응형 디자인 (모바일/데스크톱 지원)
- 🔄 **새로고침** - 최신 뉴스 업데이트

## 📦 설치

```bash
pip install -r requirements.txt
```

## 🚀 실행

```bash
python main.py
```

브라우저에서 **http://localhost:8000** 접속

## 📡 API 엔드포인트

- `GET /` - 웹 인터페이스 (카테고리 선택)
- `GET /api/categories` - 카테고리 목록 조회 (JSON)
- `GET /api/news?category={category_key}` - 카테고리별 뉴스 데이터 (JSON)
- `GET /api/news/refresh?category={category_key}` - 뉴스 새로고침
- `GET /health` - 헬스 체크

## 📁 프로젝트 구조

```
contents/
├── main.py                 # FastAPI 서버
├── crawler.py             # 스포츠 뉴스 크롤러
├── category_crawler.py    # 카테고리별 크롤러
├── requirements.txt       # 패키지 의존성
├── README.md             # 프로젝트 설명
└── templates/
    └── index.html        # 웹 인터페이스
```

## 🎯 사용 방법

1. 웹 브라우저에서 http://localhost:8000 접속
2. 원하는 카테고리 버튼 클릭
3. 해당 카테고리의 최신 뉴스 확인
4. 새로고침 버튼으로 최신 뉴스 업데이트
